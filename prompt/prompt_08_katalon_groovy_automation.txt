# YÊU CẦU ĐẦU RA
Hãy sinh cấu trúc thư mục và mã nguồn theo đúng chuẩn Katalon:
1. **Object Repository:** Liệt kê danh sách các TestObject cần tạo (tên Object và XPath/CSS tương ứng).
2. **Profiles/Variables (nếu có):** Danh sách các biến dùng chung (GlobalVariable).
3. **Custom Keyword:** Viết code Groovy (nếu kịch bản này yêu cầu thao tác phức tạp như Gen chuỗi ngẫu nhiên, format thời gian...).
4. **Test Case (Script Mode):** Cung cấp toàn bộ đoạn code Groovy thả vào tab Script của Test Case. Yêu cầu:
   - Dùng `WebUI.*` hoặc `Mobile.*`
   - Tuyệt đối không dùng Thread.sleep
   - Có comment tiếng Việt rõ ràng từng step.
   - Có bước Assert/Verify tại cuối luồng.
