# PROMPT: API AUTOMATION TEST GENERATION & EXECUTION (PROFIX)

Tôi có một bộ API Swagger (hoặc Postman Collection) và muốn thực hiện tự động hóa toàn bộ luồng kiểm thử. Hãy thực hiện cho tôi các bước sau:

1. **Phân tích:** Hãy đọc file Swagger/Collection được cung cấp để trích xuất danh sách tất cả các Endpoint và Method (GET/POST/PUT/DELETE).
2. **Thiết lập Automation:**
    - Hãy tự động chèn JavaScript (Postman Tests) vào tất cả các API này.
    - Script phải bao gồm: Status code check, Response time check (< 1000ms), Schema/Body structure check.
    - Riêng API "Login", hãy viết thêm script để lấy `token` và gán vào biến môi trường `bearerToken` để các API sau tự động sử dụng.
3. **Thực thi (Newman):** Hãy cài đặt và cấu hình Newman để chạy toàn bộ bộ test trên kết hợp với file môi trường của tôi.
4. **Báo cáo (Report & RCA):**
    - Hãy xuất báo cáo HTML cực kỳ chuyên nghiệp (dùng htmlextra) để tôi theo dõi Dashboard Pass/Fail.
    - Sau khi chạy xong, hãy thực hiện Phân tích lỗi (RCA): Liệt kê danh sách các API bị Đỏ (Fail) kèm theo nguyên nhân (4xx/500/Nghiệp vụ) và đề xuất hướng xử lý cho Developer.
